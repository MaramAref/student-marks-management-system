# Student-Marks-Management-System

A console-based C++ application for managing student records and marks.

Features

- Add new students
- Display student records
- Search for students by name
- Update student information
- Delete students
- Count total, passed, and failed students
- Find maximum and minimum marks
- Display grade estimations (A–F)
- Track system operations using a 2D array history log

Concepts Used

- Arrays
- Two-Dimensional Arrays
- Structures (Struct)
- Functions
- Modular Programming
- Input Validation
- Searching Algorithms

Technologies

- C++
- Console Application

This project was built to practice problem-solving, clean code principles, and modular program design using C++.
